## Checklist para GitHub

### Repositorio local sin repositorio remoto

- [X] Crear un nuevo repositorio en GitHub
  - El repositorio lo debemos crear sin archivos.
- [X] Conectar repositorio local
  - `git remote -v` Me aseguro de no tener conexión remota
  - `git remote add origen https...`
- [X] Actualizar repositorio remoto
  - `git push --set-upstream origin master`
